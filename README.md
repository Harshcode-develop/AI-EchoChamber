# ⚡ EchoChamber – AI Content Repurposing for Creators

**EchoChamber** is a professional AI platform engineered for modern content creators who need to scale their presence across multiple channels. It transforms long-form video content into platform-optimized, high-engagement posts for LinkedIn, X (Twitter), Instagram, YouTube, and digital blogs in seconds.

---

## ✨ Core Features

- **📹 Intelligent 1:N Repurposing**: Convert a single video source into platform-ready content for every major social channel.
- **⚡ High-Performance Generation**: AI-driven engine designed for lightning-fast turnaround without compromising quality.
- **🗣️ Dynamic Brand-Voice Awareness**: Native tone-mapping ensures all generated content sounds like you, not an AI.
- **📈 Growth Optimization**: Automatically bridges the gap between video production and text-based social reach.

---

## 🛠️ Technical Infrastructure

- **Frontend**: React 19, Vite, Tailwind CSS v4, Framer Motion, Lucide React
- **Backend**: FastAPI, Google Gemini AI (Gemini 2.0 Flash), Pydantic v2
- **Data Engine**: Supabase, Python 3.13+
- **Tooling**: Built and managed with `uv` and `npm`
